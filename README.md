# Stickman Arena Blitz

Stickman Arena Blitz is a TypeScript monorepo workspace for the game and its supporting packages.

## What this repo contains

- Shared workspace tooling at the root
- Game-related libraries and artifacts under `lib/` and `artifacts/`
- Scripts for build and validation

## Root scripts

```bash
pnpm install
pnpm run typecheck
pnpm run build
```
